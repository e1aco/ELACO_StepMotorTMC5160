# 调试经验: TMC5160 StealthChop 上电噪音

**日期**: 2026-07-23
**类型**: debug
**模块**: ela_tmc5160_usr

---

## 问题现象

断电重启后，有时发指令电机会发出很大噪音，有时又正常。不是猛转一下，而是全程运行都有噪音。

## 根因分析

TMC5160 的 StealthChop 模式在上电后会自动进行 PWM 校准。校准需要一定时间稳定，如果在校准完成前就收到运动指令，会产生噪音。

## 解决方案

在 `ela_tmc5160_init()` 末尾增加 100ms 延时，等待 PWM 校准稳定：

```c
/* 等待 StealthChop PWM 校准稳定 */
tmc5160_drv_delay_ms(100);
```

## 经验总结

1. **TMC5160 初始化时序**：使能驱动后需要等待 PWM 校准稳定（约 100ms）
2. **StealthChop 特性**：自动校准结果可能因上电状态不同而有差异
3. **调试方法**：随机性问题优先考虑初始化时序和上电状态
