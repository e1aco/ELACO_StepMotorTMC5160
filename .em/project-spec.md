# 项目规格单 — CAN PID调参功能

## 任务概述
在现有CAN协议接收命令码中新增 `0x06` 命令码，用于PID参数在线调参。

## 当前命令码
| 命令码 | 功能 |
|--------|------|
| 0x01 | 绝对位置移动(PID闭环) |
| 0x02 | 相对位置正转(PID闭环) |
| 0x03 | 相对位置反转(PID闭环) |
| 0x04 | 速度模式(开环) |
| 0x05 | 停止/刹车 |
| **0x06** | **PID调参(新增)** |

## 新增命令码 0x06 — PID调参

### CAN帧格式 (接收 ID: 0x1AA55F42)

| 字节索引 | 含义 | 说明 |
|----------|------|------|
| [0-3] | 参数值 (int32) | Q16定点数，4字节小端 |
| [4] | 命令码 | 0x06 = PID调参 |
| [5] | 电机选择 | 0x01=U1, 0x02=U2, 0x06=所有电机 |
| [6] | 参数类型 | 见下表 |
| [7] | 校验和 | 前7字节累加取低8位 |

### 参数类型定义 (字节[6])

| 值 | 参数 | Q16实际值范围 | 默认值 |
|----|------|--------------|--------|
| 0x01 | Kp (比例系数) | 0~2.0 | 0.5 (8192) |
| 0x02 | Ki (积分系数) | 0~1.0 | 0.125 (2048) |
| 0x03 | Kd (微分系数) | 0~1.0 | 0.25 (4096) |
| 0x04 | out_max (最大输出速度) | 直接整数 | 50000 |
| 0x05 | out_min (最小输出速度) | 直接整数 | -50000 |
| 0x06 | integral_max (积分限幅) | 直接整数 | 32768 |

### 反馈
调参命令执行后立即发送一次标准反馈帧(CAN ID: 0x1AA55F43)，确认参数已生效。
