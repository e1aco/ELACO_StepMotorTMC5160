# 命令: /em rq (修改需求文档)

## 功能
修改 Require/ 文件夹下的文档内容，包括 `.lnk` 指向的核心文档。

## 触发
```
/em rq <修改内容描述>              # 修改核心文档（.lnk 目标）
/em rq <文件名> <修改内容描述>      # 修改指定文件
```

## 执行流程

### /em rq <修改内容描述>（修改核心文档）
1. 查找 `Require/*.lnk` → 解析目标 → 定位核心文档 `.md` 文件
2. 根据描述修改对应内容
3. **留痕**: 在修改处标注修改时间和步骤
4. **更新索引**: 自动调用 `/em index` 更新 `require-index.md`

### /em rq <文件名> <修改内容描述>（修改指定文件）
1. 在 `Require/` 下查找匹配的文件
2. 根据描述修改对应内容
3. **更新索引**: 如行范围变化，自动更新 `require-index.md`

## 示例

```
/em rq 在CAN协议部分添加新的命令码 0x07 用于读取温度
/em rq TMC5160A_Datasheet.pdf 在寄存器表中补充 CHOPCONF 的说明
/em rq 更新实时状态：2026.07.21 完成 PID 调参测试
```

## 注意事项
- 修改核心文档时，保留原有格式和结构
- 新增内容追加到对应章节，不破坏已有内容
- PDF 文件无法直接修改，提示用户修改对应的 .md 文档
- 修改后自动更新索引，确保行范围映射准确
