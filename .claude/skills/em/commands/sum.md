# 命令: /em sum (上下文摘要)

## 功能
生成结构化摘要

## 触发
```
/em sum
```

## 执行流程

1. **【状态目录检测】** 调用 `get_state_dir()` 确定 `<STATE_DIR>`（S10-B 通用化）
2. **【读取当前状态】** 读 `<STATE_DIR>/state.md`（当前步骤 + 下一步）
3. **【读取项目规格】** 读 `<STATE_DIR>/project-spec.md`（步骤表 + 里程碑）
4. 生成结构化摘要

> 📌 **目录通用化（S10-B）**：所有 `.emv2/` 引用替换为 `<STATE_DIR>/`，
> 实际路径由 `get_state_dir()` 解析（`.em/` 优先，回退 `.emv2/`）。

- 问题：
- 进度：

### 已验证配置
- [配置列表]

1. [事项1]
2. [事项2]

### 下一步计划
-
